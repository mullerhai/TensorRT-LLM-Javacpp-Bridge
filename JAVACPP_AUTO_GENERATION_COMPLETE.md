## 🔑 关键实现细节

### 1. **Preset配置修改**
在 `TRTLLMFullConfig.java` 中添加了显式的template实例化声明：

```java
// 为JavaCPP注入显式的template实例化声明
String templateInstantiations = 
    "namespace tensorrt_llm::layers {" +
    "template class DecodingLayer<float>;" +
    "template class DynamicDecodeLayer<float>;" +
    // ... 其他10个Layer ...
    "}";

infoMap.put(new Info("").cppText(templateInstantiations));
```

### 2. **为什么之前没生成**
- 这些Layer在C++中是**template类**
- JavaCPP需要**显式的template实例化声明**（`template class DecodingLayer<float>;`）才能生成Java绑定
- 该声明在C++的.cpp文件中，但JavaCPP默认只看.h头文件
- 通过在preset中注入template实例化，强制JavaCPP识别这些类
